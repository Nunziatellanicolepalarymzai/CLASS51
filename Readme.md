--------------------------------

BINARY NUMBER 
101
=1*2^2+0*2^1+1*2^0
=4+0+1
=5(DECIMAL)
-----------------------------------

HEXADECIMAL NUMBER
AD
=A*16^1+D*16^0
=10*16^1+13*16^0
=160+13
=173(DECIMAL)
---------------------------------------

FE
=F*16^1+E*16^0
=15*16^1+14*16^0
=240+14
=254(DECIMAL)
--------------------------------------------

EX:255

STEP2-DIVIDE 15 BY 16
QUOTIENT=0
REMAINDER=15(F)
NOW CHECK THE REMAINDERS FROM DOWN TO UP
